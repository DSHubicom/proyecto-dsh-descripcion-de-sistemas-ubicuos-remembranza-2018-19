REMEMBRANZA PROJECT
________________________________________________________JGM
  
Este repositorio contiene la siguiente información:
|
|_ Audios: contiene los archivos necesarios para ejecutar el programa en el Arduino. Estos audios contienen las instrucciones y las narraciones por defecto
|
|_ Código: contiene el código del programa en Arduino, tanto el programado para la demostración del dispositivo como el destinado al producto final.
|        | Esta separación se debe a que durante la presentación de la ejecución no se efectuará una salida real a la calle, de manera que se repasan sus funcionalidades de manera simulada.
|        |
|        |CódigoDemostración: código para ejecutar la demo.
|        |CódigoRemembranza: código para ejecutar el prototipo final.
|
|_ Esquema conexiones: Contiene las imágenes que reflejan el esquema de conexiones que se sigue entre los dispositivos.
|
|_ RemembranzaProject: Contiene el código de la aplicación Android conectada con el servidor y que se encarga de enlazar las operaciones con la pulsera.
